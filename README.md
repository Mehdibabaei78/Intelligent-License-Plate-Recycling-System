## 🛠️ نصب و اجرا

### پیش‌نیازها
- Python 3.8 تا 3.11
- فایل‌های مدل در مسیر `pretrained_models/` (موجود است)

### روش اول: اجرای سریع (پیشنهادی)
```bash
cd webapp
pip install -r requirements.txt
python run.py
```
مرورگر به‌صورت خودکار روی `http://127.0.0.1:8080` باز می‌شود.

### روش دوم: با uvicorn
```bash
cd webapp
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8080
```

## 🔌 API

| متد | مسیر | کاربرد |
|-----|------|--------|
| `GET`  | `/` | صفحه‌ی اصلی |
| `GET`  | `/api/health` | وضعیت سرویس |
| `GET`  | `/api/samples` | فهرست نمونه‌ها |
| `GET`  | `/api/samples/{id}` | دانلود نمونه |
| `POST` | `/api/deblur` | دی‌بلور ساده (خروجی JPEG) |
| `POST` | `/api/deblur/full` | دی‌بلور کامل (JSON) |

### نمونه‌ی فراخوانی
```bash
curl -X POST -F "file=@test_images/322_blurred.jpg" \

```

## ⚙️ نحوه‌ی کار (پشت پرده)

پایپ‌لاین دقیقاً منطبق با اسکریپت اصلی پروژه است:

1. تصویر ورودی → خاکستری → اندازه `640×480`
2. محاسبه‌ی طیف فرکانسی **FFT**
3. تغییر اندازه به `224×224` + نرمال‌سازی
4. پیش‌بینی **زاویه** (مدل softmax، میانگین top-3) و **طول** (مدل رگرسیون)
5. ساخت **PSF** از زاویه/طول
6. **Wiener Deconvolution** برای بازیابی تصویر
7. نرمال‌سازی و خروجی JPEG
